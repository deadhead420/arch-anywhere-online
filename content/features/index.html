Arch Anywhere Features

Here you will find a full list of currently supported software and features for Arch Anywhere. Keep in mind that this list is expanding and changing as time goes on. If you have any comments, bug reports, suggestions, or ideas for future releases see below to contact developer.

The latest version of Arch Anywhere can be found on the Download page.


* Arch Linux Install:
  • Automatic mirrorlist update
  • Full base Arch Linux
  • Base development Arch Linux
  • LTS Kernel Support
  • GrSec Kernel Support
  • Fully customizable installation

* Supported partitioning methods include:
  • Full drive automatic partitioning with optional SWAP partition
  • Full drive automatic partitioning full luks on LVM encryption for root and tmp partition with optional enctypted SWAP partition
  • Manual partition (advanced users) with your choice of partitioning tool: cfdisk, fdisk, gdisk, parted

* Filesystem support:
  • ext2, ext3, ext4
  • btrfs
  • jfs
  • reiserfs

* Bootloader support:
  • GRUB (Grand Unified Bootloader)
  • SYSLINUX Bootloader
  • Support for UEFI boot
  • os-prober (Dual-Boot Support)
