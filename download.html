Arch Anywhere Github Repo


This is the official github repo of Arch Anywhere. Feel free to check out the code which went into creating Arch Anywhere. Any contrbutions, comments, suggestions, or bug reports are appreciated

Arch Anywhere consists of a modified version of the official archiso. Added to it is a custom repository allowing you to install Arch Linux, anywhere.
Built into Arch Anywhere is an installer script, allowing anyone to quickly and easily deploy an Arch Linux system regardless of their level of experience.

Arch Anywhere github repo: Arch Anywhere Scripts

If you like my project please feel free to contribute and make any changes you wish.

Arch Anywhere currently contains the following packages and package groups:
